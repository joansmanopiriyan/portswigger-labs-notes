# SQL Injection Labs – PortSwigger Web Security Academy

This section documents my hands-on practice with SQL Injection vulnerabilities from PortSwigger Web Security Academy.

The labs demonstrate different SQL injection techniques including:

- Authentication bypass
- Data extraction from hidden queries
- Database fingerprinting
- UNION-based SQL injection
- Blind SQL injection using time delays

Each report includes:

- vulnerability explanation
- payload used
- exploitation steps
- impact
- mitigation strategy

These exercises helped me understand how SQL queries interact with user-controlled input and how attackers manipulate them to retrieve unauthorized data.
