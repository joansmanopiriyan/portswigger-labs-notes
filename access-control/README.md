# Access Control Vulnerabilities – PortSwigger Web Security Academy

This section documents my hands-on practice with Access Control vulnerabilities from PortSwigger Web Security Academy.

The labs demonstrate:

- Broken administrative access controls
- Role-based privilege escalation
- Parameter-based user impersonation
- IDOR (Insecure Direct Object References)
- HTTP method manipulation
- Referer header bypass techniques
- Multi-step workflow access control weaknesses

These exercises helped me understand how improper authorization checks allow attackers to access restricted resources or perform unauthorized actions.

Each report includes:

- vulnerability description
- exploitation method
- payload used
- impact
- mitigation strategy
